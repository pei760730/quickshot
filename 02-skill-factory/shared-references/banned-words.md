## 禁用詞清單（輸出中出現即重寫）

提升、方法、思維、認知、關鍵、技巧、策略、系統化、底層邏輯、教你、分享、乾貨、秘訣、訣竅、心法、賦能、抓手、顆粒度、拉齊、對齊、打法、組合拳、閉環

## 替換表

| 禁用詞 | 替換為 | 範例 |
|--------|--------|------|
| 策略 | 做法、這樣做 | 「我的做法是___」 |
| 方法 | 怎麼做、這招 | 「怎麼做？___」 |
| 認知 | 你以為的、你想的 | 「你以為的___其實___」 |
| 思維 | 想法、怎麼想 | 「換個想法」 |
| 底層邏輯 | 真正在算的、背後的帳 | 「真正在算的是___」 |
| 關鍵 | 重點、最重要的 | 「重點是___」 |
| 技巧 | 這招、這樣弄 | 「這招很簡單」 |
| 提升 | 變好、拉高、多賺 | 「讓你多賺___」 |
| 系統化 | 一套流程、SOP | 「我有一套流程」 |
| 教你 | 告訴你、跟你說 | 「跟你說一個___」 |
| 分享 | 講、說、聊 | 「今天講___」 |
| 乾貨 | 有用的、能用的 | 「這個能直接用」 |
| 秘訣/訣竅 | 眉角、撇步 | 「眉角在這」 |
| 心法 | 想法、原則 | 「我的原則是___」 |

## 使用規則

1. 遇到禁用詞 -> 查表替換
2. 找不到替換 -> 改寫整句
3. 替換後語感要自然
