# 血包三件套（A 段落驗證 SSoT）

> version: 1.0 | last_updated: 2026-04-25
> 從 interview-navigator/evals/eval-blood-bag.md 抽出。Phase 5 退役 interview-navigator 後改放此處。
> 下游 skill：`generation/SKILL.md` mode=interview。

## 規則

每段 A 開頭必含「血包三件套」、至少 **A ≥ 2/3**：

| 血包 | 內容 | 反面（FAIL）|
|------|------|------------|
| **數字** | 真實可追溯的具體數字 / 比例 / 金額 / 頁碼等 | 「很多」「不少」「大部分」這類抽象量詞 |
| **具象敵人** | 角色（人 / 職位 / 部門）、不是品牌 / 抽象群體 | 「這個行業」「市場」「他們」這類抽象 |
| **判詞金句** | 可單獨當封面的一句話、有畫面感 | 「很黑」「不公平」這類無支撐的情緒形容 |

## 通過標準

- **3/3 滿血**：✅ PASS（最佳）
- **2/3 部分**：✅ PASS（可接受、但建議補滿）
- **0-1/3 空狠**：❌ FAIL（沒有事實支撐的情緒廢話、必須重寫）

## 附加格式檢查（A 段落）

| 項目 | 上限 |
|------|------|
| 字數 | ≤ 30 字 |
| 行數 | 2 行（建議）|
| 每行字數 | ≤ 15 字 |
| 敵人合法化 | 角色、非品牌 |
| 數字優先級 | 真實 > 合理估算 > 不寫 |

## 範例

**❌ FAIL（0/3 空狠）**：
```
「這個行業很黑」
```
缺：數字、敵人、判詞。

**✅ PASS（3/3 滿血）**：
```
「總部業務收完 80 萬，
 你連合約第 8 頁都沒看過。」
```
- 數字：「80 萬」「第 8 頁」
- 敵人：「總部業務」（角色）
- 判詞：「連合約第 8 頁都沒看過」（可當封面）

**✅ PASS（3/3 滿血、角色動作）**：
```
「中間商抽 30%，
 你以為的利潤全進他口袋。」
```
- 數字：「30%」
- 敵人：「中間商」（角色）
- 判詞：「你以為的利潤全進他口袋」（可當封面）

## 一鍵重寫格式（FAIL 時）

```
重寫段落：A?
原文案：[原句]
缺口：[數字 / 敵人 / 判詞 中缺什麼]
```
