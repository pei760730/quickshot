## 數據回填（上線後 24-72 小時）

提供 IG 截圖給 Claude → Claude 自動：
1. 讀取截圖數字
2. 計算 3秒留存率 / 完播率 / 互動率
3. 寫入 performance-patterns.json（開場公式/CTA/風險模式）

**高表現**（路徑A：3秒留存 > 70% 且 完播率 > 40%，或路徑B：觀看數 > 30萬 且 完播率 > 40%）→ 自動提取腳本規律，寫入數據大腦 [9]
**低表現**（3秒留存 < 40% 或 完播率 < 15%）→ 記錄失敗原因，保留不刪除（學習資產）
